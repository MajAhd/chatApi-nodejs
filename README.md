# NodeJs Chat Api

node.js chat api

## Available Scripts:

`npm install`

`npm run start`

`npm run start-server`

## Technologies

- Node.js && Express.js
- MongoDB
- Socket.io

## How It Work

- Uer Auth With Phone Number & Email
- Unique User Name
- Profile
- Peer To Peer Chat
- Attach Photo && Movies && File
- Group Chat
- Channel
- Contacts
